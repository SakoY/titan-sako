# Notes

This file is for freeform design notes, research findings, open questions, and discussions that don't yet belong in a formal spec or ADR. Think of it as a shared scratchpad for the project.

Move content to the appropriate spec or decision file once it stabilizes.

---

## How to Use This File

- Add dated entries for significant discussions or research sessions
- Prefix open questions with `?` so they're easy to find
- When a note resolves into a decision, move it to `decisions/` as an ADR
- When a note resolves into a requirement, move it to `specs/requirements.md`
